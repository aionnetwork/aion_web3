getId
=========

.. code-block:: javascript

    web3.eth.net.getId([callback])
    web3.bzz.net.getId([callback])
    web3.shh.net.getId([callback])

Gets the current network ID.

----------
Parameters
----------

none

-------
Returns
-------

``Promise`` returns ``Number``: The network ID.

-------
Example
-------

.. code-block:: javascript

    web3.eth.net.getId()
    .then(console.log);
    > 1
