# aion-web3-eth-abi

This is a sub package of [aion_web3][repo].

This is the abi package to be used in aion_web3.

Please read the [documentation](https://docs.aion.network/docs/web3) for more information.

## Installation

### Node.js

```bash
npm install aion-web3-eth-abi
```
